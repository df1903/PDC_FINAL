\chapter{Objetivos}

\section{Objetivo General}

Implementar y evaluar diferentes estrategias de computación concurrente y paralela para optimizar un algoritmo de scoring basado en búsqueda aleatoria, analizando su impacto sobre el desempeño computacional y la calidad de los resultados obtenidos.

\section{Objetivos Específicos}

\begin{enumerate}
    \item Implementar una solución secuencial que sirva como referencia funcional del algoritmo propuesto.

    \item Desarrollar una versión multicore utilizando mecanismos de paralelización disponibles en Python.

    \item Implementar soluciones basadas en OpenMP para aprovechar arquitecturas de memoria compartida.

    \item Implementar una versión distribuida utilizando MPI para evaluar escenarios multiproceso.

    \item Implementar una versión acelerada mediante GPU utilizando PyCUDA.

    \item Comparar experimentalmente el rendimiento de las distintas implementaciones mediante métricas de tiempo de ejecución, speedup y eficiencia.

    \item Identificar ventajas, limitaciones y oportunidades de mejora asociadas a cada paradigma de paralelización.
\end{enumerate}
