\chapter{Limitaciones del Proyecto}

Durante el desarrollo y evaluación del proyecto se identificaron diversas limitaciones.

\section{Limitaciones del Dataset}

\begin{itemize}
    \item Tamaño reducido del conjunto de datos.
    \item Uso de datos sintéticos.
    \item Escenarios limitados de validación.
\end{itemize}

\section{Limitaciones del Hardware}

\begin{itemize}
    \item Dependencia del equipo disponible.
    \item Restricción del número de núcleos.
    \item Disponibilidad de GPU compatible.
\end{itemize}

\section{Limitaciones de Paralelización}

\begin{itemize}
    \item Overhead de sincronización.
    \item Costos de comunicación MPI.
    \item Transferencias CPU--GPU.
\end{itemize}

\section{Amenazas a la Validez}

\begin{itemize}
    \item Variabilidad del entorno experimental.
    \item Carga del sistema operativo.
    \item Diferencias entre arquitecturas.
\end{itemize}
